### 演算子

- 算術演算子<br>
  | 演算子 | 演算の内容 |
  | :---: | :---: |
  | + | 加算 |
  | - | 減算 |
  | * | 乗算 |
  | / | 除算 |
  | // | 商 |
  | % | 剰余 |
  | ** | べき乗 |
  | := | 代入 |

  ```
  b = (a := 2) * 2
  ```
  出力結果は**2*2の答え**

- 論理演算子<br>
  | 演算子 | 動作 | Rubyの場合 |
  | :---: | :---: | :---: |
  | and | 論理積 | && |
  | or | 論理和 | \|\| |
  | not | 否定 | != |
  | is | | aとbの変数のIDが等しいかどうか判断 |
  | in | | 左に右の文字列が含まれるか判断 |
