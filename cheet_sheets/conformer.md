### 型

- Pythonの組み込み型<br>
  | 型 | 型名 |
  | :---: | :---: |
  | int | 整数型 |
  | float | 浮動小数点型 |
  | str | 文字列型 |
  | bool | 真偽値型 |

  - 型を調べる
    ```
    type(変数や値)
    ```

  - 別の型に変換
    ```
    型(値または変数)
    ```

  - フォーマット文字列（変数や値の埋め込み）
    ```
    price = 550
    print(f'この商品は{price}円です')
    この商品は550円です
    ```
